⚙️ Travel Flow Negócios

Configure a conexão com o seu servidor backend

Conexão com o servidor

Endereço completo da pasta onde os arquivos PHP do backend estão instalados. Não inclua barra no final.

Chave definida em API_KEY no db.conf do servidor. Necessária para todos os usuários.

Acesso de administrador opcional

Chave definida em ADMIN_KEY no db.conf do servidor. Quando preenchida, habilita a aba ⚙️ Campos no painel lateral, permitindo adicionar, reordenar e remover campos do formulário. Deixe em branco para usuários comuns.

Como instalar o backend

  1. Baixe os arquivos PHP e o schema.sql em github.com/rmacari/travelflow-negocios
  2. Envie todos os arquivos para uma pasta no seu servidor (ex: /httpdocs/travelflow-negocios/)
  3. Importe o schema.sql no seu banco de dados MySQL. Em banco já existente, execute também as migrações necessárias, incluindo migrate_v3.sql.
  4. Crie o arquivo db.conf na mesma pasta: 
    DB_HOST=localhost
DB_NAME=seu_banco
DB_USER=seu_usuario
DB_PASS=sua_senha
ALLOWED_ORIGIN=https://travelflow.tur.br
API_KEY=chave_para_todos_os_usuarios
ADMIN_KEY=chave_somente_para_administradores
  5. Gere chaves seguras e diferentes para API_KEY e ADMIN_KEY:
    openssl rand -hex 32
  6. Preencha os campos acima e clique em Salvar configurações
