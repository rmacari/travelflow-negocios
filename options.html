⚙️ Zap Negócios

Configure o servidor do mini CRM universal para WhatsApp Web, leads, negócios, tarefas e lembretes

Conexão com o servidor

Endereço completo da pasta onde os arquivos PHP do backend estão instalados. Não inclua barra no final.

Chave definida em API_KEY no db.conf do servidor. Necessária para a instalação da extensão.

Login do usuário

As permissões vêm do papel definido no servidor: viewer, editor, admin ou owner.

Como instalar o backend

  1. Baixe os arquivos PHP e o schema.sql em github.com/rmacari/zap-negocios
  2. Envie todos os arquivos para uma pasta no seu servidor (ex: /httpdocs/zap-negocios/)
  3. Importe o schema.sql no seu banco de dados MySQL. Em banco já existente, execute também as migrações necessárias, incluindo migrate_v4.sql, migrate_v5.sql, migrate_v6.sql e migrate_v7.sql.
  4. Crie o arquivo db.conf na mesma pasta: 
    DB_HOST=localhost
DB_NAME=seu_banco
DB_USER=seu_usuario
DB_PASS=sua_senha
ALLOWED_ORIGIN=https://travelflow.tur.br,https://web.whatsapp.com
API_KEY=chave_da_instalacao
SETUP_KEY=chave_para_criar_o_primeiro_owner
  5. Gere chaves seguras e diferentes para API_KEY e SETUP_KEY:
    openssl rand -hex 32
  6. Abra setup_owner.php no navegador para criar o primeiro owner, depois faça login nos campos acima
