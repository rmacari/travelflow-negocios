Configure a conexão com o seu servidor backend

Conexão com o servidor

Endereço completo da pasta onde os arquivos PHP do backend estão instalados no seu servidor. Não inclua barra no final.

Mesma chave definida na variável API_KEY do arquivo db.conf no servidor. Usada para autenticar as operações de gerenciamento de campos.

Como instalar o backend

  1. Baixe os arquivos PHP e o schema.sql em github.com/macari/travelflow-negocios
  2. Envie todos os arquivos para uma pasta no seu servidor (ex: /httpdocs/travelflow-negocios/)
  3. Importe o schema.sql no seu banco de dados MySQL
  4. Crie o arquivo db.conf na mesma pasta com o conteúdo: 
    DB_HOST=localhost
DB_NAME=seu_banco
DB_USER=seu_usuario
DB_PASS=sua_senha
ALLOWED_ORIGIN=https://travelflow.tur.br
API_KEY=sua_chave_secreta
  5. Gere uma chave secreta segura com o comando:
    openssl rand -hex 32
  6. Cole a URL do servidor e a API Key nos campos acima e clique em Salvar configurações
